NetMonitor est un système client-serveur permettant de surveiller en temps réel les ressources système de plusieurs machines sur un réseau local et de partager des fichiers entre elles.

Fonctionnalités

Surveillance des ressources système

Collecte des métriques système en temps réel (CPU, mémoire, disque, réseau)

Interface web de visualisation

Tableau de bord interactif montrant l'état des machines connectées

Partage de fichiers

Téléversement et téléchargement de fichiers via l'interface web

Technologies utilisées

Backend

Python avec architecture modulaire et programmation orientée objet

Communication réseau

Sockets TCP/IP, sélection non-bloquante (select())

API Web

Flask pour exposer une API REST et servir l'interface web

Frontend

HTML/CSS/JavaScript avec Chart.js pour les visualisations
